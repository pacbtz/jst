# jst: JSON Value Types via S7 Classes

Provides S7 classes for representing JSON value types (null, boolean,
number, string, array, object). Typed JSON values can be constructed
with json_null(), json_boolean(), json_number(), json_string(),
json_array(), and json_object(), and serialised to JSON text via
format() / print().
